keyupとkeydownとkeypressの違い

発火する順番は、keyDown → keyPress → keyUp

keydownとkeyupは、全てのキーに対応するが、keypressはシフトキーなど文字でない入力は反応しない。

keyPressは機能が削除される予定の為、非推奨。
