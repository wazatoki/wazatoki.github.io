hugo開発サーバーへlocalhost外からのアクセス

目次

hugo server -Dで開発サーバーを起動するとlocalhost外からアクセスできない。


$>hugo server --baseURL="192.168.0.1" --bind="192.168.0.1" --port=8080

で起動するとhttp://192.168.0.1:8080でアクセスできる。
